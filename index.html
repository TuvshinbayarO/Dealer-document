Installation Application requirement

Алхам 1

Node суулгах

Та Node болон NPM систем дээрээ суулгасан эсэхийг шалгах хэрэгтэй. Хэрэв тэдгээрийг суулгаагүй бол эндээс татаж авах боломжтой.

Алхам 2

Node modules суулгах

Node modules хамааралтай node modules суулгахын тулд та dealerapp хавтсанд терминалыг нээж дараахь зүйлийг бичих хэрэгтэй. тушаал. Үүнд хэсэг хугацаа зарцуулагдаж магадгүй. 

      npm install

Алхам 3

Starting DealerApp

DealerApp програмыг эхлүүлж байна Програмыг эхлүүлэхийн тулд. Та терминалд дараах коммандыг оруулах шаардлагатай. Хэдэн секундын дараа Android эсвэл Ios Simulator Ажиллана 

      npx expo start

Алхам 4

Aab build

Та aab build гаргахдаа дараах коммандыг терминалд бичиж өгнө 

                    cd android && ./gradlew clean && ./gradlew bundleRelease

Алхам 5

Apk build

Та apk build гаргахдаа дараах коммандыг терминалд бичиж өгнө 

                    npm run android -- --mode="release"
