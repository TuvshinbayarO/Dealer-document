Installation Application requirement

Танилцуулга

Dealer App нь expo SDK48 болон React Vesion 18.2.0 дээр last байдлаар update хийсэн!
Дэмжигдэх үйлдийн систем: Android version 5+ & Ios version 13+

Алхам 1

Node суулгах

Та Node болон NPM систем дээрээ суулгасан эсэхийг шалгах хэрэгтэй. Хэрэв тэдгээрийг суулгаагүй бол эндээс татаж авах боломжтой.

Алхам 2

Node modules суулгах

Node modules хамааралтай node modules суулгахын тулд та dealerapp хавтсанд терминалыг нээж дараахь зүйлийг бичих хэрэгтэй. тушаал. Үүнд хэсэг хугацаа зарцуулагдаж магадгүй. 

      npm install

Алхам 3

Starting DealerApp

DealerApp програмыг эхлүүлж байна Програмыг эхлүүлэхийн тулд. Та терминалд дараах коммандыг оруулах шаардлагатай. Хэдэн секундын дараа Android эсвэл Ios Simulator Ажиллана 

      npx expo start

Алхам 4

Aab build

Та aab build гаргахдаа дараах коммандыг терминалд бичиж өгнө 

                    cd android && ./gradlew clean && ./gradlew bundleRelease

Алхам 5

Apk build

Та apk build гаргахдаа дараах коммандыг терминалд бичиж өгнө 

                    npm run android -- --mode="release"

Алхам 7

Test Flight apk гаргах

TestFlight нь Apple-ийн официал аппликейшн тестлэгийн платформ бөгөөд бүх iOS байгуулагдсан аппликейшнуудыг зөвхөн зарим хэрэглэгчидэд туршлага хийж, санал хүсэлтүүд авч турших боломжийг олгоно. Та Expo ашиглан аппликейшн бэлтгэхэд, тестлэгийн алхам рүү орохоор эхлээд хэрхэн бүртгүүлэх, турших, засварлах талаар мэдээллээ авахыг харуулж байна. Дараах зааврыг дагаж мөн биш болгоно уу: 

                    expo build:ios

TestFlight-руу аппликейшн оруулах: Expo бэлтгэсэн проектын бүртгэл дээр хамгийн сүүлийн аппликейшны байгууллагын нэрийг дамжуулж, аппликейшныг тестлэгийн алхам рүү оруулах шаардлагатай. Хамгийн сүүлийн аппликейшн нэг хоногийн дотор TestFlight-д харуулагдана. Тестлэгийн санал хүсэлтүүд авах: TestFlight рүү орж, тестлэгийн санал хүсэлтүүдийг хяналтад оруулж, аппликейшнд гарч ирсэн алдаануудыг шалгана. Тэдгээрийг бүртгэсэн ажлын хэрэглэгчид аппликейшн болон тестлэгийн алхам рүү орж, туршиж болно.

Алхам 8

PlayStore publish

1. Google Play Developer Console-руу нэвтрэх: Эхлээд Google Play Developer Console-руу орох шаардлагатай. Та Google Play Developer болон хэрэглэгчийн мэдээллийг ашиглан нэвтэрнэ.
2. Аппликэйшний мэдээлэл оруулах: Дэлгэрэнгүй мэдээллийг ашиглан аппликэйшний мэдээлэл оруулна. Энэ хэсгээр аппликэйшний нэр, верс, тайлбар, зураг, шинэчлэл, багцын мэдээлэл, төрөл болон бусад мэдээллийг оруулна.
3. AAB файл оруулах: Таны аппликэйшний кодыг агуулсан APK (Android Package) файлыг оруулна. Энэ файл нь аппликэйшнийг суулгаж, ажиллуулах код, шошго, зураг, зургийн тодорхой мэдээлэл болон бусад боломжуудыг агуулсан бөгөөд хэмжээ нь таны аппликэйшний төрлийн дагуу өөрчилнө.
4. Туршилтын мэдээлэл оруулах: Аппликэйшний хяналтын тайлбар, зураг, видео, холбоос, туршилтын агуулга болон бусад мэдээллийг оруулна. Эдгээр мэдээллийг хэрэгтэй хэлбэрээр оруулж болно.
5. Зарлага болон төлбөрийн мэдээлэл оруулах: Та Play Store-д таны аппликэйшний хэрэглэгчид хэрхэн төлбөр төлөх боломжтойг тодорхойлох зарлага, төлбөрийн арга харгалзах мэдээллийг оруулна.
6. Техникийн мэдээлэл оруулах: Аппликэйшний ажиллагааны зориулалтыг тайлбарлан тодорхойлох техникийн мэдээлэл, аппликэйшний хэрэглэгчдэд хэрхэн ажилладаг талаарх мэдээллийг оруулна.
7. Аппликэйшний хяналтын төлбөрийг тохируулах: Та Play Store-д аппликэйшнийг худалдаалж авах төлбөрийн болон хяналтын төлбөрүүдийг тохируулж болно.
8. Аппликэйшнийг хэрхэн ашиглахыг тайлбарлах: Та аппликэйшнийг хэрхэн ашиглах талаар хэрэглэгчиддээ илтгэл үзүүлэх тайлбарлага болон зурвас оруулах боломжтой.
9. Нийтлэл, зурвас болон тусламж оруулах: Таны аппликэйшний талаарх нийтлэл, зурвас, тусламж, сэтгэгдэл болон бусад мэдээллийг оруулж болно.
10. Аппликэйшнийг нэвтрүүлэх хувь хүний мэдээллийг тохируулах: Та аппликэйшнийг ашиглах хэрэгтэй хувь хүний мэдээллийг тохируулж болно.
11. Туршилтын тайлбар, зурвас, төлбөр болон зөвлөгөө оруулах: Таны аппликэйшнийг хэрхэн суулгах, ашиглах, хэрэглэгчдэд зөвлөгөө үзүүлэх талаарх тайлбар, зурвас, төлбөр болон зөвлөгөөг оруулж болно.
12. Аппликэйшнийг шинэчлэх болон байхгүй болгох: Та аппликэйшнийг шинэчлэх, засах эсвэл 

                    AAB file хуулах
